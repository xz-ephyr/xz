> **Date:** 2026-06-23
> **Icon (Hugeicons):** `TreePalmIcon`
> **Currently:** 6-line placeholder stub in `src/pages/WikiPage.tsx`

---

## Concept

A **personal knowledge base / wiki** — Markdown pages written by the user or generated by the AI, organized hierarchically. Think Notion + Obsidian lite.

---

## Possible Features

| Feature | Description |
|---------|-------------|
| **Markdown pages** | Create, edit, delete wiki pages in Markdown. Rendered with the same Markdown renderer as chat messages (`MarkdownMessage.tsx`). |
| **Folder/tree hierarchy** | Pages organized in a folder tree (like the Projects section or VS Code explorer). Drag-and-drop reordering. |
| **Full-text search** | Search across all wiki page titles and content. |
| **AI-assisted creation** | "Turn this chat into a wiki page" — button in chat to save assistant's answer as a wiki page. |
| **Wiki from AI tool** | The AI can use a `write_to_wiki` tool to create/update wiki pages during conversation. |
| **Page history** | Version history for each wiki page (reuse the `artifact_versions` table pattern). |
| **Linking** | `[[Wiki Links]]` syntax for cross-page references. |
| **Tags** | Tag pages for filtering. |
| **Export (stretch)** | Export wiki as a zip of markdown files. |

## Data Model (speculative)

```ts
interface WikiPage {
  id: string;
  title: string;
  content: string;
  path: string;                // "folder/subfolder/page"
  parentId: string | null;
  tags: string[];
  createdAt: number;
  updatedAt: number;
}

interface WikiFolder {
  id: string;
  name: string;
  parentId: string | null;
  order: number;
}
```
