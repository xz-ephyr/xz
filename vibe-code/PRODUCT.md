# Product

A local-first AI developer desktop app built with Tauri. It combines a code editor, terminal, AI routing, SQL access, and agent workflows in one small desktop workspace.

## Principles
- local-first
- BYOK
- small binary
- simple contributor experience
- modular monolith
- no cloud dependency for core usage

## Audience
- solo developers
- small teams
- power users
- contributors who want a readable codebase

## Core surfaces
- code editor
- terminal
- AI chat
- SQL workspace
- file explorer
- agent workflows
- settings
- local memory
