# Design

## Final stack
- Tauri
- Rust backend for OS integration
- TypeScript frontend
- Vite
- Svelte
- CodeMirror 6
- xterm.js
- Vercel AI SDK
- SQLite
- OS credential store for secrets

## Avoid
- Electron
- Monaco Editor
- LangChain-style complexity
- Python runtime in the main app
- vector database at the core
- Docker dependencies
- Antigravity SDK as a core runtime dependency

## UI language
- constrained widths
- soft neutral surfaces
- low-radius geometry
- fast micro-motion
- clear typography hierarchy
